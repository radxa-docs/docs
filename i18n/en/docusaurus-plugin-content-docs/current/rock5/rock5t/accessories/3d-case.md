# Third-Party 3D-Printed Cases

This page lists third-party, community-designed 3D-printed cases for the Radxa ROCK 5T. These models make it easy to quickly print an enclosure and protect the board.

> Note: The following cases are all designed by third-party community authors and are not officially affiliated with Radxa. Please evaluate risks yourself before printing and using them.

## Model 1: ROCK 5T Face Screw Case

- MakerWorld model page:
  - https://makerworld.com/en/models/2298370-rui-sha-radxa-rock-5t-mian-luo-si

Key features:

- Mounting holes aligned with Radxa ROCK 5T
- Dedicated face-screw case design for ROCK 5T
- Suitable for desktop or rack-mount scenarios

---

Printing suggestions:

- Prefer heat-resistant, dimensionally stable materials (such as PETG, ABS, etc.)
- Before printing, read the model author's notes on MakerWorld (recommended layer height, infill, supports, etc.)
- During installation, avoid applying excessive force on cables, connectors, and components
