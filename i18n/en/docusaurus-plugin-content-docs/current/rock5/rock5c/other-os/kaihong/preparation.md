# Preparation

## Development setup

<Tabs queryString="target">

<TabItem value="necessary item" label="Necessary Items">

### Power supply

We recommend a 5V/3A Type-C power adapter for ROCK 5C.

:::tip Power options

The ROCK 5C board only supports 5V input and is PD compatible. A 3A or higher current is recommended to ensure stable operation with peripherals.

Recommended power supplies:

- [Radxa PD 30W Power Adapter (recommended)](https://radxa.com/products/accessories/power-pd-30w)

- Standard Type-C power adapter (5V input, PD compatible, 3A or higher recommended)

:::

### Storage

The system can boot from a microSD card or an eMMC module.

- microSD card: capacity >= 8GB, for booting from microSD
- eMMC module: capacity >= 8GB, for booting from eMMC

:::tip Recommended accessories

- [Radxa eMMC Module](https://radxa.com/products/accessories/emmc-module)

:::

### Card reader

- microSD card reader: for flashing images to microSD
- Radxa eMMC reader: for flashing images to eMMC

:::tip Recommended accessories

eMMC modules support both USB cable flashing and flashing with an eMMC reader. When using an eMMC reader, the process is the same as flashing a microSD card.

- [Radxa eMMC/UFS Reader](https://radxa.com/products/accessories/emmc-ufs-module-reader)

:::
