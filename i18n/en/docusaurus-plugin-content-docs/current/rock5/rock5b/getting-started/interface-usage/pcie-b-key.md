# PCIe B Key (only for ROCK 5B+)

PCIe B Key can be used to insert a wwan module to enable cellular network connectivity.

## Supported Accessories

[quectel-lte-ec25](../../../../accessories/peripherals/quectel-lte-ec25.md)

## Third-party Accessory Issue Records

1. When using EM7430/EM7455 modules, there is a prompt that the SIM card cannot be recognized. This may be related to the SIM card recognition pin. According to forum user feedback, shielding one of the recognition pins can make it work normally:
[Forum Link](https://forum.radxa.com/t/rock-5b-sim-card-design-flaw/24530)
