# Antenna Interface

The Radxa Cubie A7Z comes with onboard WiFi 6 and Bluetooth 5.4, featuring 1 antenna interface for connecting an external antenna to enhance wireless communication performance. The product includes 1× Wi-Fi / Bluetooth antenna (model: AC019 Dual Band FPC Antenna) in the box.

:::info Interface Location
For the specific location of the antenna interface, please refer to the interface description section in the [Hardware Interface](./hardware-info) tutorial!
:::

## Included Antenna Specifications

| Item           | Specification                          |
| -------------- | -------------------------------------- |
| Antenna Model  | AC019 Dual Band FPC Antenna            |
| Connector Type | 1st generation IPEX                    |
| Dimensions     | 42mm × 11.9mm                         |
| Cable Length   | 50mm                                   |
| Frequency      | Dual-band (Wi-Fi 2.4GHz / 5GHz)       |
| Operating Temp | -70°C ~ +80°C                          |

## Usage Guide

Connect the antenna to the corresponding antenna interface. The antenna interface uses 1st generation IPEX connector.

After completing the antenna installation, you can configure the wireless network, check the wireless signal strength, and test network bandwidth performance according to the [Wireless Network](../system-config/wifi-usage) guide.
