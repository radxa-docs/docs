# Radxa Taco

## Product Overview

The Radxa Taco is a highly expandable carrier board designed for the [**Raspberry Pi Compute Module 5 (CM5)**](https://www.raspberrypi.com/products/compute-module-5/), suitable for edge computing and lightweight server scenarios requiring multi-storage, multi-network, and stable power supply capabilities.

Building on the CPU / memory / graphics and multimedia performance of the CM5, the Taco provides comprehensive engineering expansion capabilities:

- Storage Expansion: microSD card slot, M.2 M Key 2280 slot, 5x SATA interfaces
- Network Expansion: Gigabit Ethernet, 2.5G Ethernet, M.2 E Key 2230 slot
- Peripheral Expansion: 2x USB 3.2 Gen1 Type-A interfaces, fan, RTC and other interfaces
- Power & Maintenance: 12V DC power jack / header, RPI BOOT button, Type-C flash interface

This enables the Taco to rapidly build deployable NAS, gateway, industrial control, and local AI inference platforms while maintaining CM5 ecosystem compatibility.

## Application Scenarios

- **Lightweight NAS / Home Media Center**: Build large-capacity local storage using 5x SATA and M.2 storage interfaces.
- **Dual-Port Edge Gateway**: Gigabit + 2.5G Ethernet ports suitable for software routers, bypass gateways, and network service integration.
- **Industrial & Device Control**: Meet long-term operation requirements through GPIO, RTC, fan, and stable 12V power supply.
- **Local AI & Vision Inference Node**: Combine CM5 computing power to deploy edge recognition, detection, and data collection tasks.
- **Development & Teaching Platform**: Rich interface types, suitable for Linux system, driver, and hardware debugging experiments.
