# Install Heatsink

This guide introduces how to install the heatsink onto the Radxa Taco. Users can select part of the accessories for installation based on their needs.

## Package List

Raspberry Pi CM5 and SATA hard drives need to be purchased separately. The image shows the Radxa Taco accessory list.

Note: The accessory list includes the heatsink and brackets for securing SATA hard drives.
