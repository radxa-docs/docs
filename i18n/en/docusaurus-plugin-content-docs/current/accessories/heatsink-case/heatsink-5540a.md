# Radxa Heatsink 5540A

## Product Overview

The Radxa Heatsink 5540A is a passive heatsink designed for Radxa CM3 and CM5.

:::tip Version Notes

The Radxa Heatsink 5540A has two versions: the older version is silver, and the newer version is black.

:::

</div>

## Specifications

- Material: Aluminum alloy
- Dimensions: 55 x 40 x 10 mm
- Compatible with:
  - Radxa CM3
  - Radxa CM5

## Installation Guide

This section uses Radxa CM3 as an example to demonstrate how to install the 5540A heatsink.

### Package Contents

- 1x Heatsink
- 1x Thermal pad
- 4x CM2 x 4 mm screws

### Install the Antenna (Optional)

If you are using an external antenna, install it on the CM3 before installing the heatsink.

<div style={{textAlign: 'center'}}>
    <img src="/en/img/accessories/heatsink-case/heatsink-5540a/install-ante.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### Apply the Thermal Pad

Place the thermal pad on the SoC of the CM3.

<div style={{textAlign: 'center'}}>
    <img src="/en/img/accessories/heatsink-case/heatsink-5540a/install-silicon.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### Secure the Heatsink

Use 4 CM2 x 4 mm screws to fasten the heatsink from the bottom side through the CM3 mounting holes.

<div style={{textAlign: 'center'}}>
    <img src="/en/img/accessories/heatsink-case/heatsink-5540a/install-heatsink.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### Mount the Assembly to the CM3 IO Board

Install the CM3 with the heatsink attached onto the CM3 IO board.
