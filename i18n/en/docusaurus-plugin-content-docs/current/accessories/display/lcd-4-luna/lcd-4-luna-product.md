# Radxa Display 4 Luna

## Product Overview

<div style={{textAlign: 'center'}}>
    <img src="/en/img/accessories/display/display-4-luna-view.webp" style={{width: '100%', maxWidth: '1200px'}} />

The Radxa Display 4 Luna is a high-quality IPS display with touch support, designed for rapid deployment across multiple boards and scenarios.

Using a single USB Type-C cable, it carries both video and touch signals, supports extended and mirrored display modes, works plug-and-play on Linux, Windows, and macOS hosts, and is compatible with Radxa, Raspberry Pi, and other common single-board computers. Key highlights:

- 4.0-inch IPS LCD
- 720 × 720 resolution
- 5-point capacitive touchscreen
- USB Type-C connection (video + touch)
- Supports extended / mirrored display modes
- Cross-platform support (Linux / Windows / macOS)

## Specifications

- Size: 4.0 inch
- Resolution: 720 × 720
- Display type: IPS
- Touch type: 5-point capacitive
- Interface: USB Type-C (video + touch)
- Display modes: Extended / Mirrored
- OS support: Linux / Windows / macOS
- Power: USB Type-C
- Enclosure (optional): MagSafe-style case
- Raspberry Pi support: Raspberry Pi 3 / 4 / 5
- Power consumption: typical < 1 W
- Brightness: 350 nits
- Viewing angle: 178° (horizontal / vertical)
- Contrast ratio: 800:1
