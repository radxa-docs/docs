Rsetup is a system configuration tool for Radxa OS that helps users easily configure their system, including system settings, hardware settings, software settings, network settings, user settings, language settings, and more.

## About Rsetup

Run the `rsetup` command in the terminal to start the Rsetup tool.

:::tip User Guide

- Select option: The corresponding option will be highlighted
- Confirm selection: Press `Enter`
- Cancel selection: Press `ESC`
- Navigate options: Use `Up`, `Down`, `Left`, `Right` arrow keys
- Multiple selection interface: Press `Space` to select, `Enter` to confirm; a `*` will appear next to selected options to indicate they are enabled
  :::

<NewCodeBlock tip={`radxa@${props?.board ?? 'device'}$`} type="device">

```
sudo rsetup
```

</NewCodeBlock>
