The BIOS (Basic Input/Output System) is the motherboard firmware that runs first when the board powers on. It wakes up hardware devices, performs self-tests, and finally boots the operating system, acting as the bridge between hardware and software.

## BIOS Overview

:::tip Navigation tips

- Arrow keys: Move between menu items
- Enter: Open or confirm the highlighted option
- ESC: Return to the previous level or exit the setup screen

Mastering these shortcuts makes it easier to navigate all BIOS pages.

:::
