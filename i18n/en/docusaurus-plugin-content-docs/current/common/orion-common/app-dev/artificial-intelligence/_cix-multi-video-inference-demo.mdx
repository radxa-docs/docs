**CIX NPU Multi-Channel Video Real-time Object Detection System** is a multi-process YOLO multi-channel video real-time object detection system based on the Radxa Orion O6 / O6N NPU. It uses the NPU for YOLO model inference, supports multi-channel video parallel processing, and is suitable for video surveillance, intelligent transportation, and other scenarios.

- Core Features: NPU Hardware Acceleration, Multi-process Architecture, V4L2 Hardware Decoding, YOLO Post-processing, COCO 80-class Object Detection
- Version Notes: This project provides two working modes — Queue Buffer Mode (Recommended) and Adaptive Speed Mode. The former pursues maximum throughput, while the latter is suitable for resource-constrained scenarios.

```txt

├── models/                      # Pre-trained model directory
```

### Parameter Description

| Parameter       | Default         | Description                                            |
| --------------- | --------------- | ------------------------------------------------------ |
| `--video_paths` | Required        | Input video file path or directory (supports multiple) |
| `--model_path`  | Required        | CIX quantized model file path                          |
| `--conf_the`    | 0.5             | Confidence threshold                                   |
| `--nms_the`     | 0.45            | NMS IoU threshold                                      |
| `--target_size` | 640             | Model input size                                       |
| `--output_dir`  | ./output_videos | Output directory in headless mode                      |
| `--timeout`     | 0               | Runtime duration limit (seconds), 0 means unlimited    |
