Open-source AI models for various common domains have been precompiled for different NPU hardware architectures and are available for download.
Users can directly download and deploy them using [QAIRT](qairt-sdk#qairt) or [QAI-APPBUILDER](qai-appbuilder).

Model types include:

- Computer Vision Models

- Speech Models

- Generative Models

- Multimodal Models

## Download Links

- [**Qualcomm AI HUB**](https://aihub.qualcomm.com/models)

  :::tip
  For QCS6490 models, select `Qualcomm QCS6490(Proxy)`

  For QCS9075 models, select `Qualcomm QCS9075(Proxy)`, `Qualcomm QCS8550(Proxy)`, and `Snapdragon X Elite`
  :::
