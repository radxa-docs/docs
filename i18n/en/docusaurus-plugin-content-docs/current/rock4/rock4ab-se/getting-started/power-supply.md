# Power Supply Options

ROCK 4A/4B/4A+/4B+ supports multiple power supply methods, smart power adapters, and fixed voltage inputs:

<Tabs queryString="model">

    <TabItem value="ROCK 4A/4B">
        - USB PD 2.0 support for USB Type-C PD 2.0, 9V/2A, 12V/2A, 15V/2A, 20V/2A
        - Qualcomm® Quick Charge™ 2.0 support for QC3.0/2.0 adapters, 9V/2A, 12V/1.5A
        - Power adapter's USB C power port can provide fixed voltage from 6V to 24V
        - 5V power from GPIO pins 2 and 4
        - ROCK 4B Gigabit Ethernet port (near USB) supports PoE with additional Radxa 23W PoE HAT
    </TabItem>

    <TabItem value="ROCK 4A+/4B+/4SE">
        - USB Type-C™ PD 2.0, 9V/2A, 12V/2A
        - Qualcomm® Quick Charge™ 2.0 QC3.0/2.0 adapters, 9V/2A, 12V/1.5A
        - Power adapter with fixed voltage from 6V to 12V on USB C power port
        - 5V power from GPIO pins 2 and 4
        - ROCK 4B+ Gigabit Ethernet port (near USB) supports PoE with additional Radxa 23W PoE HAT
    </TabItem>

</Tabs>

:::tip

Recommended to use the official 30W adapter [Radxa PD 30W Adapter](../../../accessories/power/pd_30w)

:::
