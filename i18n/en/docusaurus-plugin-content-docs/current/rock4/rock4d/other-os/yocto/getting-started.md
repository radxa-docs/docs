# Getting Started

This tutorial aims to guide you through getting started with the Yocto Project, building and developing Linux systems based on the vendor version of the Yocto Project.

## Build System

Refer to the [Build System](./build-system) documentation.

## Install System

Refer to the [Install System](./install-system) documentation.

## Boot System

Install the corresponding system boot medium to the ROCK 4D, connect a display, and finally power the ROCK 4D using a 5V Type-C power adapter to start the system.
