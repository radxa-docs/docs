# Radxa AICore AX-M1

## Product overview

The Radxa AICore AX-M1 is a high-performance M.2 accelerator module based on the AXERA AX8850 SoC. It features high compute capability and power efficiency, designed for edge AI computing and inference applications.

The Radxa AICore AX-M1 integrates a multi-core high-performance CPU and a high-compute NPU with excellent multimedia processing capability, providing efficient and flexible hardware support for various edge AI scenarios.

## Key features

- **High-performance SoC platform** — Powered by the AX8850 SoC with an octa-core Cortex-A55 CPU up to 1.5 GHz, delivering strong general-purpose compute and multi-tasking capability.

- **Powerful AI compute** — Integrated high-performance NPU supporting up to 24 TOPS@INT8 neural network inference to meet high-speed inference needs of complex AI models.

- **Professional-grade multimedia processing**

  - Supports up to 8K@30fps video input and image processing for high-resolution video analytics.
  - Equipped with a video processor supporting H.264/H.265 HD encoding/decoding for diverse streaming and intelligent video analytics scenarios.

- **Hardware compatibility** — Supports multiple host platforms including Intel, AMD, and Rockchip.

- **OS compatibility** — Compatible with multiple operating systems including Ubuntu, Debian, and CentOS.

- **Standardized interface and compatibility** — Standard M.2 M Key 2280 size, compatible with mainstream industrial and embedded boards.
