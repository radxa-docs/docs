# Pulsar2 Toolchain

The Pulsar2 toolchain helps developers compile models from mainstream AI frameworks (PyTorch, TensorFlow, Caffe, ONNX, etc.) into axmodel format models that can run inference on the Radxa AICore AX-M1.

</div>

- For installation and usage, see the [Pulsar2 online documentation](https://pulsar2-docs.readthedocs.io/zh-cn/latest/).
- For the latest release, visit [Huggingface Pulsar2](https://huggingface.co/AXERA-TECH/Pulsar2).
