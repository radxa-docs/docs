# Firmware Update

Radxa Linkr supports firmware update directly via the web and flashing firmware via USB cable.

If the web firmware update fails or the device cannot work properly, you can refer to the firmware flashing method to flash the latest firmware onto Linkr.

## Firmware Download

- GitHub Releases: [Radxa Linkr](https://github.com/radxa-linkr/linkr/releases)

- Baidu Pan Download: [Radxa Linkr: Extraction code 1234](https://pan.baidu.com/s/1AZgLUDaU8hJKhZjqTT_sMQ?pwd=1234)

Note: The web firmware update downloads a file in `<system-image>.bin` format, while the USB flashing downloads a file in `<system-image>.img` format.

## Firmware Update

It is recommended to use the web firmware update method first. If the Linkr firmware update fails or the device cannot work properly, then use the USB flashing method to update the firmware.

<Tabs queryString="update-firmware">

<TabItem value="Web Firmware Update">

Web firmware update requires the host computer to be able to access the Linkr-controlled device. We need to upload and update firmware in the management interface at [linkr.now](https://linkr.now/).

- Access the controlled device interface

Enter the management interface of the controlled device to update the Linkr firmware:

① : Go to the Settings option

② : Select `System Update` for firmware update

③ : Upload the firmware file that needs to be flashed

<TabItem value="USB Firmware Flashing">

Flashing firmware via USB needs to be performed on a Windows platform.

The computer needs to install the [DriverAssistant](https://dl.radxa.com/tools/windows/DriverAssitant_v5.14.zip) driver and download the [SocToolKit](https://github.com/radxa-linkr/linkr/releases/download/Tools/SocToolKit_V1.94.zip) tool.

- Driver Installation

Download and extract the [DriverAssistant](https://dl.radxa.com/tools/windows/DriverAssitant_v5.14.zip) package, find `DriverInstall.exe` and run it as administrator, then click the `Install Driver` option to install the driver.

<div style={{ textAlign: "center" }}>
  <img
    src="/img/common/tools/rkdevtool/rkdevtool-driver.webp"
    style={{ width: "50%", maxWidth: "700" }}
  />
</div>

- Enter Maskrom Mode

Press and hold the user button on Linkr, use a USB-C data cable to connect the USB-C data/power port on the side of Linkr to the computer, and release the user button after the connection is successful.

Detailed steps:

① : Press and hold the user button on Linkr

② : Use a USB-C to USB-A data cable to connect Linkr to the computer

③ : Release the user button on Linkr

<div style={{ textAlign: "center" }}>
  <img
    src="/img/linkr/linkr-maskrom.webp"
    style={{ width: "100%", maxWidth: "1200" }}
  />
</div>

- Tool Download

Download and extract the [SocToolKit](https://github.com/radxa-linkr/linkr/releases/download/Tools/SocToolKit_V1.94.zip) package, find `SocToolKit.exe` and run it as administrator.

- Select Product Chip

① : Select the `RV1106` chip model

② : Click `OK` to confirm the selection

<div style={{ textAlign: "center" }}>
  <img

</div>

- Update Firmware

① : Confirm that the software has recognized the Maskrom device

② : Click `Firmware ...` to select the firmware to be flashed

③ : Click the `Upgrade` option to update the firmware
