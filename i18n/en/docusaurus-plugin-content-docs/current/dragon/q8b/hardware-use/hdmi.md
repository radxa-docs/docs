# HDMI Output

The Radxa Dragon Q8B has an onboard HDMI output port, which can connect to monitors, TVs, and other devices for video output.

## Hardware Connection

Connect an HDMI cable to the HDMI port on the Dragon Q8B, and connect the other end to the HDMI port of a monitor, TV, or other display device.

:::tip
Some monitors may have compatibility issues. Try using another monitor for testing.

You can refer to the [Hardware Information](./hardware-info) guide to locate the corresponding hardware interface.
:::
