# EDL Button

The Radxa Dragon Q8B has an onboard EDL button for entering EDL mode. EDL mode is an emergency download mode for Qualcomm processor devices, mainly used for emergency recovery, firmware flashing, or device unlocking.

## User Guide

If the EDL button is detected as pressed while the motherboard is powering on, the motherboard will enter EDL mode.

Detailed steps:

1. Press and hold the EDL button
2. Connect a 20V Type-C power adapter compatible with the PD protocol
3. Release the EDL button

Normally, the motherboard will enter EDL mode.
