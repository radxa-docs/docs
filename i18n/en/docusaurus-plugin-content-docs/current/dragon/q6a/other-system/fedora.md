# Fedora

Fedora is a community-supported Linux distribution sponsored by Red Hat. It is known for providing the latest open-source software, cutting-edge technologies, and innovative features, serving as the upstream project for RHEL (Red Hat Enterprise Linux).

## Key Features

- **Cutting-Edge Technology**: First to adopt the latest Linux kernels and software packages
- **Open Source Purity**: Committed to free and open-source software principles
- **Innovation Platform**: Many new technologies are first tested and deployed in Fedora
- **Strong Community**: Active global community of developers and users
- **Multiple Desktop Environments**: Supports GNOME, KDE, Xfce, and more

## Installation Instructions

1. Download the Fedora image for Radxa Dragon Q6A
2. Use an image writing tool to flash the image to an SD card or UFS storage
3. Insert the storage medium into the device and boot
4. Follow the on-screen prompts to complete initial configuration
