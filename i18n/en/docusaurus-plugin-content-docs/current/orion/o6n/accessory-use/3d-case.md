# Third-Party 3D-Printed Cases

This page lists third-party, community-designed 3D-printed cases for the Radxa Orion O6N. These models make it easy to quickly print an enclosure and protect the board.

> Note: The following cases are all designed by third-party community authors and are not officially affiliated with Radxa. Please evaluate risks yourself before printing and using them.

## Model 1: Screwless Snap-Fit Case

- MakerWorld model page:
  - https://makerworld.com.cn/zh/models/1861722-rui-sha-radxa-orion-o6n-3dda-yin-wai-ke-mian-luo-s

Key features:

- Mounting holes aligned with Radxa Orion O6N
- Screwless design: tool-less snap-fit structure for easy installation and removal
- Dust-protective design, suitable for long-term installation on racks, workbenches, and similar scenarios

## Model 2: Ultra-Light Minimal Protective Case

- MakerWorld model page:
  - https://makerworld.com.cn/zh/models/1865300-radxa-orion-o6nwai-ke

Key features:

- Very simple and lightweight structure, easy and quick to print
- Suitable as a daily protective enclosure for desktop or lab environments
