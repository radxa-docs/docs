# Computer on Module Family

Radxa's System on Module (SoM) product line covers multiple form factors, allowing users to flexibly select and expand based on application scenarios and existing platforms. The comparison table below summarizes the form factors, compatible ecosystems, and typical applications of each series to help you make quick decisions:

| Series | Form Factor & Package                                                 | Compatible Ecosystem / Standard            | Key Features                                                                                                                | Typical Applications                                                        |
| ------ | --------------------------------------------------------------------- | ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| CM     | Raspberry Pi CM4/CM5 compatible module with board-to-board connectors | Raspberry Pi CM4 ecosystem                 | Compatible with Raspberry Pi CM4/CM5 in size and interfaces, easy to upgrade existing Raspberry Pi compute module solutions | Performance upgrades for existing CM4 devices, rapid solution validation    |
| NX     | Jetson Nano/Orin NX compatible SoM with SODIMM edge connector         | NVIDIA Jetson Nano/Orin Nano/Orin NX, etc. | Integrated high AI computing power, designed for edge AI inference, compatible with Jetson hardware ecosystem               | Edge AI gateways, machine vision, intelligent robots, industrial inspection |

Through these series, Radxa provides a unified software ecosystem and diverse hardware form factors for different stages from development validation to mass production.
