# FAQ

## ROCK 2A Frequently Asked Questions

### Issue: My ROCK 2A stops booting after running for a period of time

If your ROCK 2A suddenly fails to boot after running normally for a period of time, try the following troubleshooting steps:

#### 1. Power Check
- Recommend using a **5V / 3A** power adapter
- If the indicator light is on, power supply is normal; if not, there may be a hardware issue

#### 2. System Troubleshooting
- If the system fails to boot despite normal power supply → use serial console to read system logs for diagnosis
- You can seek help by posting log information via doc comments or GitHub Issue
