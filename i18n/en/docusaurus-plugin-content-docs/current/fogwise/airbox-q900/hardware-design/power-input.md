# DC Power Input

The Fogwise® AIRbox Q900 is powered by a 12V DC power adapter.

## Interface Specifications

| Parameter             | Specification                              |
| --------------------- | ------------------------------------------ |
| Interface Type        | DC Jack                                    |
| Jack Size             | 2.5mm inner diameter, 5.5mm outer diameter |
| Polarity              | Center positive                            |
| Voltage               | 12V                                        |
| Current (Recommended) | ≥ 5.4A                                     |
| Compatibility         | Supports PD power adapter                  |

## User Guide

Use the original or compatible 12V DC power adapter to power the mainboard.
