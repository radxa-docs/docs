## User Guide

When the AIRbox Q900 is powered on, briefly press the power button to display the system shutdown option. If no action is taken within 60 seconds, the system will automatically shut down.

Note: After shutdown, you must unplug and reconnect the power adapter before powering on again.

:::tip

If the AIRbox Q900 is installed in a metal enclosure, you can use a pin, toothpick, or similar tool to press the power button.

:::
