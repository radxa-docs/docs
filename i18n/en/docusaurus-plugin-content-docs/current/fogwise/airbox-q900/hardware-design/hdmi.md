# HDMI Interface

The Radxa Fogwise® AIRbox Q900 features one standard HDMI 2.0 interface for display connectivity, supporting up to 4K@60fps (3840×2160@60Hz) resolution output.

## Usage Guide

Connect the AIRbox Q900's HDMI interface to a display using an HDMI cable.

:::tip Compatibility

Some displays may not work properly when connected to the AIRbox Q900. Try connecting to a different display if you encounter any issues.

:::
