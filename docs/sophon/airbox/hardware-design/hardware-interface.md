## 电源接口

**支持 20V / 65W / 3.25A 的电源输入。**

## 调试串口

使用 USB Type-A 转 Type-C 数据线连接整机 Debug 口和 PC 端，波特率 115200。

## MicroSD

支持热插拔。目前推荐三星和闪迪的SD卡。

## 有线网口

一个是WAN口，一个是LAN口。
