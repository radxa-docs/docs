# 镜像下载

瑞莎团队以 Debian 为基础开发了 Radxa OS 作为官方操作系统。

如果你想尝试其他镜像，请参阅[第三方镜像](#第三方镜像)。

:::note
注意：许多第三方镜像未经过官方测试，可能存在一些问题。
:::

:::tip
提示：我们将会在[瑞莎论坛](https://forum.radxa.com/)发布每次版本发布的相关信息。
:::
