# 固件更新

瑞莎灵控（Radxa Linkr）支持网页直接更新固件和线刷固件。

对于网页更新固件失败或者无法正常使用，可以参考线刷固件的方式给灵控刷入最新固件。

## 固件下载

- GitHub 发布：[瑞莎灵控（Radxa Linkr）](https://github.com/radxa-linkr/linkr/releases)

- 百度网盘下载：[瑞莎灵控（Radxa Linkr）：提取码 1234](https://pan.baidu.com/s/1AZgLUDaU8hJKhZjqTT_sMQ?pwd=1234)

说明：网页更新固件下载 `<system-image>.bin` 格式的文件，线刷固件下载 `<system-image>.img` 格式的文件。

## 固件更新

优先推荐使用网页更新固件的方式进行固件更新，若灵控固件更新失败或者无法正常使用，再使用线刷固件的方式更新固件。

<Tabs queryString="update-firmware">

<TabItem value="网页更新固件">

网页更新固件需要主机可以正常访问灵控控制的设备，我们需要在 [linkr.now](https://linkr.now/) 的控制界面进行固件的上传与更新

- 访问被控设备界面

进入被控设备的控制界面进行灵控固件更新：

① : 进入设置选项

② : 选择 `System Update` 进行固件更新

③ : 将需要刷入的固件文件放进去

- 更新固件

将固件文件放进去后，系统会自动上传、安装固件，最后重启设备后生效。

<TabItem value="线刷更新固件">

以线刷的方式更新固件需要在 Windows 平台下进行。

电脑需要安装 [DriverAssistant](https://dl.radxa.com/tools/windows/DriverAssitant_v5.14.zip) 驱动和下载 [SocToolKit](https://github.com/radxa-linkr/linkr/releases/download/Tools/SocToolKit_V1.94.zip) 工具。

- 驱动安装

下载 [DriverAssistant](https://dl.radxa.com/tools/windows/DriverAssitant_v5.14.zip) 压缩包并解压，找到 `DriverInstall.exe` 并以管理员身份运行，点击 `Install Driver` 选项进行驱动安装。

<div style={{ textAlign: "center" }}>
  <img
    src="/img/common/tools/rkdevtool/rkdevtool-driver.webp"
    style={{ width: "50%", maxWidth: "700" }}
  />
</div>

- 进入 Maskrom 模式

按住灵控上的用户按键，使用 USB-C 数据线连接灵控的侧边 USB-C 数据/供电接口和电脑，连接成功后松开用户按键。

详细步骤：

① : 按住灵控上的用户按键

② : 使用 USB-C 转 USB-A 数据线连接灵控和电脑

③ : 松开灵控上的用户按键

<div style={{ textAlign: "center" }}>
  <img
    src="/img/linkr/linkr-maskrom.webp"
    style={{ width: "100%", maxWidth: "1200" }}
  />
</div>

- 工具下载

下载 [SocToolKit](https://github.com/radxa-linkr/linkr/releases/download/Tools/SocToolKit_V1.94.zip) 压缩包并解压，找到 `SocToolKit.exe` 并以管理员身份运行。

- 选择产品芯片

① : 选择 `RV1106` 芯片型号

② : 点击 `OK` 确认选项

<div style={{ textAlign: "center" }}>
  <img
    src="/img/linkr/usb_update_firmware_01.webp"
    style={{ width: "100%", maxWidth: "1200" }}
  />
</div>

- 更新固件

① : 确认软件已经识别到 Maskrom 设备

② : 点击 `Firmware ...` 选择需要刷入的固件

③ : 点击 `Upgrade` 选项更新固件

<div style={{ textAlign: "center" }}>
  <img
    src="/img/linkr/usb_update_firmware_02.webp"
    style={{ width: "100%", maxWidth: "1200" }}
  />
</div>

- 更新固件成功

软件的右边信息输出区提示 `... OK` 说明固件更新成功。
