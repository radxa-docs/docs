# 产品概述

瑞莎灵控（Radxa Linkr）是一款远程控制工具，基于 Web 的键盘鼠标远程显示方案打造，通过 HDMI 采集被控主机画面，经 USB 模拟键鼠输入，让你在任何有浏览器的终端上远程查看、操作目标电脑——无需安装专用客户端。

设备体积小巧，支持有线网络、Wi-Fi 与热点多种联网方式，并内置 Tailscale、Wake on LAN、访问令牌等高级能力，适合家庭实验室、远程运维与嵌入式开发等场景。

## 产品定位

瑞莎灵控定位于**轻量级、开箱即用的基于键盘鼠标的远程显示解决方案**：

- **即插即用**：HDMI + USB Type-C 连接被控主机，浏览器打开管理页面即可使用
- **Web 优先**：远程画面、键鼠控制、网络与系统管理统一在 Web 控制台完成
- **场景友好**：兼顾桌面 PC、开发板、无头服务器等设备的远程维护需求

与传统机架式远程控制设备或纯软件远程桌面方案相比，瑞莎灵控在成本、部署复杂度和使用门槛之间取得平衡，适合个人开发者、小团队及教育场景。

## 目标用户

| 用户群体 | 典型场景 |
| --- | --- |
| 开发者与创客 | 远程调试开发板、无显示器环境下的系统安装与排障 |
| 家庭实验室用户 | 管理家中 NAS、软路由、备用 PC，无需额外显示器和键盘 |
| 运维与技术支持 | 远程协助用户操作电脑、抓取画面、发送控制指令 |
| 教育与培训 | 课堂演示、实验环境统一远程访问与管控 |

## 核心价值

- **随时随地访问**：局域网、互联网（配合 Tailscale）均可连接管理界面
- **完整远程操控**：视频采集 + 键鼠注入，体验接近本地操作
- **一体化管理**：网络配置、固件升级、虚拟磁盘挂载、API 集成均在同一 Web 界面
- **可扩展集成**：访问令牌支持截图、控制等公开 API，便于自动化与监控

## 产品外观

瑞莎灵控采用紧凑型设计，正面与侧面分布有 HDMI 接口、USB Type-C 接口、FN 功能键及状态指示灯。具体外观以实物为准，可参考产品包装或官方资料中的示意图。

## 接口与连接

| 接口 | 用途 |
| --- | --- |
| HDMI 公口 | 接入被控主机，采集视频信号 |
| USB Type-C（OTG） | 供电与数据传输（模拟被控机的鼠标、键盘等 HID 设备） |
| USB Type-C（网口） | 有线以太网通信 |
| Wi-Fi | 连接无线网络 |
| 热点 | 设备作为 AP 共享网络 |

### 连接说明

1. **HDMI 公口**：连接被控主机的 HDMI 视频输出，用于采集画面。
2. **USB Type-C（OTG）**：同时承担供电与数据通道，向被控机提供键鼠等 USB HID 功能。
3. **USB Type-C（网口）**：百兆有线网络，用于瑞莎灵控自身的网络通信。
4. **Wi-Fi**：接入现有无线网络，免布线部署。
5. **热点**：瑞莎灵控可作为热点，供其他终端连接以访问管理界面或共享网络。

典型接线方式：HDMI 连接被控机视频输出，OTG Type-C 连接被控机 USB 口（供电 + 键鼠），网口 Type-C 或 Wi-Fi 接入网络。

## LED 指示灯

设备 **FN 键** 两侧各有一颗指示灯：

- **靠近 HDMI 一侧**：视频（HDMI）状态灯
- **远离 HDMI 一侧**：系统状态灯（RGB，通过颜色表示运行状态）

### 系统状态灯

| LED 状态 | 说明 |
| --- | --- |
| 蓝色闪烁 | 系统启动中 |
| 蓝色常亮 | 系统正常运行 |
| 绿色常亮 | 正在被使用中（有活跃连接） |
| 白色闪烁 | 正在升级固件 |
| 白色常亮 | 正在重置系统 |

### HDMI / 视频状态灯

| LED 状态 | 说明 |
| --- | --- |
| 灭 | 无视频信号 |
| 亮 | 有视频信号 |
| 闪烁 | 视频链路异常或错误 |

通过观察指示灯，可在不打开 Web 界面的情况下快速判断设备与视频链路状态。

## FN 按钮

| 操作 | 功能 |
| --- | --- |
| **长按 ≥ 10 秒** | 重置系统（将所有配置恢复到出厂默认状态） |
| **短按一次** | 设备识别（用于 [linkr.now](https://linkr.now) 上的设备发现功能） |

:::caution 注意
长按 FN 键重置将清除所有本地配置（含网络、账号、Tailscale 绑定等），操作前请确认已备份必要信息。
:::
