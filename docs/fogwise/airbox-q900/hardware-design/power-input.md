# DC 电源输入接口

Fogwise® AIRbox Q900 使用 12V DC 电源适配器进行供电。

## 接口规格

| 参数         | 规格                   |
| ------------ | ---------------------- |
| 接口类型     | DC 插座                |
| 接口尺寸     | 内径 2.5mm、外径 5.5mm |
| 极性         | 内正外负               |
| 电压         | 12V                    |
| 电流（推荐） | ≥ 5.4A                 |
| 兼容性       | 支持 PD 电源适配器供电 |

## 使用指南

使用原装或符合规格的 12V DC 电源适配器给主板进行供电。

## 注意事项

- 使用不符合规格的电源可能导致设备损坏

- 确保电源功率充足，避免因供电不足导致系统不稳定
