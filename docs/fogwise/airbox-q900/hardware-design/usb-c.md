# USB Type-C 接口

瑞莎 Fogwise® AIRbox Q900 板载 1 个 USB Type-C 接口，主要用于系统日志查看和命令行交互。

## 使用指南

使用 USB Type-C 数据线连接主板的 USB Type-C 接口和 PC 的 USB 接口。

:::tip 使用说明

更多串口登录的使用方法请参考 [串口登录](../system-use/uart-login.md) 教程。

:::
