# HDMI 接口

瑞莎 Fogwise® AIRbox Q900 板载 1 个 标准 HDMI2.0 接口，用于连接显示器，最高支持 4K@60fps（3840×2160@60Hz）分辨率输出。

## 使用指南

使用 HDMI 数据线连接 AIRbox Q900 的 HDMI 接口和显示器。

:::tip 兼容性

部分显示器可能连接 AIRbox Q900 后无法正常显示，可尝试连接其他显示器测试。

:::
