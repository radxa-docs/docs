# 风扇接口

瑞莎 Fogwise® AIRbox Q900 板载风扇接口，用于连接风扇。

:::tip 金属外壳

瑞莎 Fogwise® AIRbox Q900 金属外壳内置 PWM 风扇。

:::

## 硬件连接

将金属外壳的 PWM 风扇接口连接到瑞莎 Fogwise® AIRbox Q900 的风扇接口。
