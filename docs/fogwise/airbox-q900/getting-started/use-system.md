# 使用系统

主要介绍如何使用系统，包括启动系统、系统用户信息、以及有屏模式和无屏模式的使用建议。

## 启动系统

使用 12V DC 电源适配器给 Fogwise® AIRbox Q900 通电，若系统启动介质存在正常的系统，系统将自动启动。

## 系统信息

首次启动系统，你需要使用默认的用户名和密码登录。

默认凭据如下：

| 用户名 | 密码  |
| :----: | :---: |
| radxa  | radxa |

## 使用系统

主要给有屏和无屏模式的用户提供一些建议，方便大家更好的上手使用我们的主板。

### 有屏模式

有屏模式是指使用 HDMI 数据线连接主板和显示器，通过显示器显示系统界面和使用鼠标和键盘进行图形化界面操作。

该方式操作系统比较简单，可以根据系统选项进行配置。

### 无屏模式

无屏模式是指不连接显示器，通过串口登录系统或者 SSH 登录系统。

该方式使用命令行进行交互，更多依赖命令行进行系统的操作。
