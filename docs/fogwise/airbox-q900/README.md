# Fogwise® AIRbox Q900

<Tabs queryString="target">

<TabItem value="Fogwise® AIRbox Q900" label="Fogwise® AIRbox Q900">

## 产品介绍

Fogwise® AIRbox Q900 是一款嵌入式人工智能微型服务器，算力高达 100TOPS@INT8，支持多种精度（INT8、FP16/BF16、FP32），支持私有 GPT、文本到图像等多种主流人工智能模型部署，并配备散热铝合金外壳，可在恶劣环境中部署。
