# Radxa Products

Welcome to Radxa Product Center! Radxa offers five product lines — single-board computers (mini PCs / motherboards), compute modules, network computing devices, AI acceleration modules, and edge computing platforms — covering general computing, embedded development, industrial control, edge AI, networking, and multimedia applications.

- **Single-Board Computers / Mini PCs / Motherboards**: Ideal for general computing, multimedia, IoT, and edge AI applications
- **Compute Modules**: Designed for embedded systems, industrial control, and customized products
- **Network Computing Devices**: Built for network equipment, routers, and edge computing
- **AI Acceleration Modules**: Edge AI inference modules supporting major deep learning frameworks
- **Edge Computing Platforms**: Complete development ecosystem with industrial-grade design for diverse edge AI deployments
