CIX AI Model Hub 仓库托管在魔搭社区上，访问地址 [cix ai_model_hub](https://modelscope.cn/models/cix/ai_model_hub_25_Q3)

由于仓库体积较大，建议避免全量克隆。克隆必要的运行文件后，您可以根据左侧导航栏选择感兴趣的模型，查阅其专属文档并进行针对性部署，以节省存储空间。
:::

## 下载必要文件

### 环境准备

确保已安装最新版本的 ModelScope SDK

<NewCodeBlock tip="Linux PC" type="PC">

```bash
pip install --upgrade modelscope
```

</NewCodeBlock>

### 使用脚本下载

创建一个 Python 脚本（例如 download.py），核心逻辑如下：

:::info
下载完成之后需要将文件同步到板端。
:::

<NewCodeBlock tip="Linux PC" type="PC">

```python
from modelscope.hub.snapshot_download import snapshot_download

# --- 参数配置 ---
model_id = 'cix/ai_model_hub_25_Q3'
target_folder = [
    'utils/*',
    'requirements.txt'
]
save_dir = './ai-model-hub/'

# 执行下载
snapshot_download(
    model_id,
    allow_file_pattern=target_folder,
    local_dir=save_dir
)
```

</NewCodeBlock>

### 板端创建虚拟环境并安装相关依赖

<NewCodeBlock tip="O6 / O6N" type="device">

```bash
python3 -m venv .venv
pip3 install -r requirements.txt
```

</NewCodeBlock>

## 下载整个仓库

使用 git 下载（请确保安装 git-lfs）

<NewCodeBlock tip="Linux PC" type="PC">

```bash
mkdir ai-model-hub && cd ai-model-hub
git lfs install
git clone https://www.modelscope.cn/cix/ai_model_hub_25_Q3.git
```

</NewCodeBlock>
