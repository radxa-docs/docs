**CIX NPU 多路视频实时目标检测系统** 是基于 瑞莎 Orion O6 / O6N NPU 的多进程 YOLO 多路视频实时目标检测系统。它利用 NPU 进行 YOLO 模型推理，支持多路视频并行处理，适用于视频监控、智能交通等场景。

- 核心特点：NPU 硬件加速、多进程架构、V4L2 硬件解码、YOLO 后处理、COCO 80 类目标检测
- 版本说明：本项目提供两种工作模式——队列缓冲模式（推荐）和自适应速度模式，前者追求最高吞吐量，后者适合资源受限场景

需要提前配置好相关环境。

  :::

### 参数说明

| 参数            | 默认值          | 说明                               |
| --------------- | --------------- | ---------------------------------- |
| `--video_paths` | 必填            | 输入视频文件路径或目录（支持多个） |
| `--model_path`  | 必填            | CIX 量化模型文件路径               |
| `--conf_the`    | 0.5             | 置信度阈值                         |
| `--nms_the`     | 0.45            | NMS IoU 阈值                       |
| `--target_size` | 640             | 模型输入尺寸                       |
| `--output_dir`  | ./output_videos | 无头模式下输出目录                 |
| `--timeout`     | 0               | 运行时长限制（秒），0 表示无限制   |
