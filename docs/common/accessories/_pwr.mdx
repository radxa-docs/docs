### 软件重启

#### 在图形界面关机或重启

- 点击左下角的图标，选择**离开**选项。

- 你将能够看到系统功能选项。

- 你可以选择你需要的功能**（睡眠、重启、关机）**。

#### 通过终端控制

在桌面上打开终端命令（或使用Ctrl+Alt+T快捷键）。

**重启**  
输入以下命令并按回车键，随后输入密码。
如果密码正确，它将重新启动系统。
如果错误，请重新输入正确的密码。

```bash
sudo reboot
```

**关机**  
输入以下命令并按回车键，随后输入密码。
如果密码正确，它将关闭系统。
如果错误，请重新输入正确的密码。

```bash
sudo shutdown now
```

### 电源键

如图所示，箭头所指的地方就是电源键。

- 当设备处于关闭状态时，按下电源键，设备将打开。工作时，电源绿灯会一直亮着，状态蓝灯会闪烁。

<img src={props.model_img} alt="pwr" style={{ width: "80%" }} />

- 当设备开启时，按下电源键，屏幕会弹出用户状态选择。你可以选择睡眠、重启、关机或登出。当你关闭设备或者设备处于睡眠时，电源绿灯和状态蓝灯将被关闭。
