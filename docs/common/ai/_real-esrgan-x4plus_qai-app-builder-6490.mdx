此文档讲述如何使用 QAI AppBuilder python 库在瑞莎 Dragon Q6A 上运行 [Real-ESRGAN-x4plus](https://aihub.qualcomm.com/models/real_esrgan_x4plus?domain=Computer+Vision&useCase=Super+Resolution&chipsets=qualcomm-qcs6490-proxy)
图像超分模型。

## 安装 QAI AppBuilder

:::tip
请根据 [**QAI AppBuilder 安装方法**](qai-appbuilder)
:::

## 运行示例

### 安装依赖

<NewCodeBlock tip="Device" type="device">

```bash
pip3 install requests tqdm qai-hub py3_wget opencv-python torch
```

</NewCodeBlock>
