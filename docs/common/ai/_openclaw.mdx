[OpenClaw](https://github.com/openclaw/openclaw) 是由 Peter Steinberger 开发的开源 AI Agent 项目,
可以在任意设备，任意 OS 上部署的个人 AI 助手。

## 安装 OpenClaw

使用安装脚本安装 OpenClaw

<NewCodeBlock tip="Device" type="device">

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

</NewCodeBlock>

## 配置 OpenClaw

:::tip
使用脚本安装时就会出现配置方式。如果想重新配置请运行以下命令

<NewCodeBlock tip="Device" type="device">

```bash
openclaw onboard --install-daemon
```

</NewCodeBlock>
:::
