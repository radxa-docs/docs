此文档讲述使用 [QAI AppBuilder](../qai-appbuilder) Python API 利用 Qualcomm® Hexagon™ Processor (NPU) 推理 [VIT](https://aihub.qualcomm.com/models/vit)
目标识别模型。

**示例支持设备**

| 设备       | SoC     |
| ---------- | ------- |
| Dragon Q6A | QCS6490 |

## 安装 QAI AppBuilder

:::tip

1. 请根据 [**QAI AppBuilder 安装方法**](../qai-appbuilder) 安装 QAI AppBuilder。

2. 请根据 [**配置 QAIRT 环境变量**](../qai-appbuilder#配置-qairt-环境变量) 配置 QAIRT 环境变量。
   :::

## 运行示例

### 安装依赖


</NewCodeBlock>
