:::tip
系统默认不安装 NPU 设备环境，若有 NPU 使用需求，请参考此文档进行 NPU 设备环境安装
:::

NPU 环境配置清单

- [fastrpc](https://github.com/qualcomm/fastrpc)
- /dev/fastrpc-adsp
- /dev/fastrpc-cdsp
- /dev/fastrpc-cdsp-secure
- /usr/lib/dsp libraries
