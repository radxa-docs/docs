Balena Etcher 是一个跨平台且，用户界面友好的镜像文件烧写工具，我们推荐你使用它。

1. 下载 Etcher 并安装。

<Tabs queryString="Platform">
<TabItem value="Windows">

请下载 [balenaEtcher-Setup-1.18.11.exe](https://github.com/balena-io/etcher/releases/download/v1.18.11/balenaEtcher-Setup-1.18.11.exe)。下载完成后，无需进行额外操作，双击即可打开使用。

</TabItem>
<TabItem value="Linux">

请下载 [balena-etcher_1.18.11_amd64.deb](https://github.com/balena-io/etcher/releases/download/v1.18.11/balena-etcher_1.18.11_amd64.deb)。下载完成后，请在终端执行以下命令进行安装：

```
sudo dpkg -i balena-etcher_1.18.11_amd64.deb
```

</TabItem>
</Tabs>

2. 打开 Etcher

3. 打开 Etcher，单击 `Flash from file` 以选择需要写入的镜像。

   <Image
     src="/img/common/etcher/#model#-step2.webp"
     model={props.model}
     fallback="rock5a"
   />

4. 点击 `Select target` 以选择设备，请注意小心选择。

   <Image
     src="/img/common/etcher/#model#-step3.webp"
     model={props.model}
     fallback="rock5a"
   />

5. 点击 `Flash!` 开始写入，然后等待写入进度条完成。

   <Image
     src="/img/common/etcher/#model#-step4.webp"
     model={props.model}
     fallback="rock5a"
   />

6. 当写入镜像成功时，Etcher 将会显示 `Flash Complete!`。
   <Image
     src="/img/common/etcher/#model#-step5.webp"
     model={props.model}
     fallback="rock5a"
   />

**如果刷写操作系统镜像错误, 请手动再试一次。**
