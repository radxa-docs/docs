BIOS（Basic Input/Output System）是主板上的固件，通电后首先运行，用于唤醒硬件、完成自检并引导操作系统启动。它是连接硬件与操作系统的“桥梁”。

## BIOS 总览

:::tip 操作说明

- 方向键：在菜单之间移动
- Enter：进入或确认当前选项
- ESC：返回上一层或退出设置界面

掌握上述快捷键，便于在各功能页面中快速导航。

:::
