BIOS（Basic Input/Output System）是计算机主板上的固件程序，通电后第一段运行的代码，用于唤醒硬件、完成自检，并引导操作系统启动。换言之，BIOS 是连接硬件与操作系统的“桥梁”。

## BIOS 总览

:::tip 操作说明

- 方向键：在菜单之间移动。
- Enter：进入或确认当前选项。
- ESC：返回上一层或退出设置界面。

掌握上述快捷键，有助于在接下来的各功能页面中快速导航。

:::
