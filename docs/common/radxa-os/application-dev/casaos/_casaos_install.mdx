## CasaOS 特点

CasaOS 是一款专为家庭用户设计的轻量级云操作系统，具有简单易用、安全可靠、扩展性强等特点。

- 简单易用

CasaOS 采用直观的 Web 界面设计，通过简单的拖拽操作，即可实现文件和照片的上传、下载以及备份，无需复杂配置，即使是非技术用户也能轻松上手。

- 安全可靠

在数据传输和存储方面采用了多种加密和保护措施，确保用户数据的安全。同时，它还支持快照功能，可在出现意外情况时迅速恢复到之前的状态，自主掌握数据，避免数据泄露风险。

- 扩展性强

支持各种常见的存储设备和服务，如 USB 闪存盘、SD 卡、硬盘等，同时也支持各种云存储服务，用户可以根据自己的需求灵活扩展存储空间。

- 应用生态丰富

内置应用商店，提供 Nextcloud、HomeAssistant、AdGuard 等精选应用，支持一键安装 Docker 容器应用，可轻松安装和管理应用，覆盖媒体管理、智能家居、网络优化等场景，满足用户多样化的需求。

- 高度自定义

用户可以根据自己的喜好定制桌面主题，甚至可以通过 API 接口开发个性化插件，打造符合自己需求的智能家居管理系统。
