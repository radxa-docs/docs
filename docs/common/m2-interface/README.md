# M.2 接口协议说明

## 概述

M.2 接口是一种用于连接各种扩展设备的通用接口标准。Radxa 产品中常见的 M.2 接口类型包括 M Key、B Key 和 E Key，每种类型支持不同的协议和兼容不同的设备。

## M.2 接口类型与协议支持

### M Key (M 型接口)
- **主要用途**: 高速存储设备
- **支持的协议**:
  - PCIe (PCI Express) - 主要用于 NVMe SSD
  - 通过转接板可支持 SATA (需使用 M.2 M Key 转 SATA 转接板)
- **兼容设备**:
  - NVMe SSD (M.2 2280 规格)
  - 通过转接板可使用 SATA SSD/HDD
  - 部分 AI 加速卡
- **物理特征**: 接口右侧有 5 个引脚缺口

### B Key (B 型接口)
- **主要用途**: 通信模块
- **支持的协议**:
  - USB 2.0/3.0
  - PCIe (通常为 PCIe 2.0 x1)
  - 部分产品可能支持 UART
- **兼容设备**:
  - 4G/5G 移动网络模块
  - 部分无线通信模块
- **物理特征**: 接口左侧有 6 个引脚缺口

### E Key (E 型接口)
- **主要用途**: 无线网络模块
- **支持的协议**:
  - PCIe (通常为 PCIe 2.0 x1)
  - USB 2.0
  - SATA
  - SDIO
  - 部分产品支持 UART 和 PCM
- **兼容设备**:
  - WiFi 6/6E 无线网卡
  - 蓝牙模块
  - 部分无线通信模块
- **物理特征**: 接口左侧有 4 个引脚缺口

## 兼容产品选择指南

### 选择存储设备
1. **确定接口类型**: 检查产品是否有 M Key 接口
2. **确认协议支持**: 
   - 如果支持 PCIe: 选择 NVMe SSD
   - 如果支持 SATA: 选择 SATA SSD (需确认具体产品是否支持)
3. **确认物理规格**: 通常是 M.2 2280 (22mm宽×80mm长)

### 选择无线网络模块
1. **确定接口类型**: 通常是 E Key
2. **确认协议支持**: 通常支持 PCIe 和 USB
3. **确认物理规格**: 通常是 M.2 2230 (22mm宽×30mm长)

### 选择移动网络模块
1. **确定接口类型**: 通常是 B Key
2. **确认协议支持**: 通常支持 USB 和 PCIe
3. **确认 SIM 卡支持**: 检查产品是否有 SIM 卡槽

## 常见问题

### Q: 如何知道我的 Radxa 产品支持哪种 M.2 接口？
A: 请参考具体产品的文档，在"参数配置"或"接口说明"部分查找 M.2 相关信息。

### Q: 我可以将 M Key 的 SSD 插入 B Key 接口吗？
A: 不可以。M Key 和 B Key 的物理接口不同，无法插入。请确保设备与接口类型匹配。

### Q: 为什么我的 NVMe SSD 不被识别？
A: 可能的原因:
1. SSD 与接口类型不匹配 (需要 M Key 接口)
2. SSD 规格不支持 (如产品只支持特定长度的 SSD)
3. 需要安装相应的驱动程序

### Q: 如何安装 M.2 设备？
A: 一般步骤:
1. 将设备以 30° 角插入 M.2 插槽
2. 轻轻按下设备使其平贴
3. 使用螺丝固定
4. 安装驱动程序 (如果需要)
