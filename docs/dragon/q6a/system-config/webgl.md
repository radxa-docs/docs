# WebGL 使用指南

## 安装 chromium-browser

<NewCodeBlock tip="radxa@dragon-q6a$" type="device">

```bash

sudo apt install chromium-browser

```

</NewCodeBlock>

## 运行 chromium-browser

打开浏览器 chromium-browser。

输入网址 https://webglreport.com/ 或 https://get.webgl.org/，可以查看到 WebGL 支持情况。
