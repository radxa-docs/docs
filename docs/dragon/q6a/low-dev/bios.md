# BIOS 说明

BIOS（Basic Input/Output System）是计算机主板上的一个固件程序，是计算机通电后运行的第一段代码，它的核心任务是唤醒硬件、检查身体、并引导操作系统正常工作。

BIOS 是连接计算机硬件与操作系统的“桥梁”。
