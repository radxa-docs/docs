# Fedora

Fedora 是一个由社区支持的 Linux 发行版，由 Red Hat 赞助开发。它以提供最新的开源软件、前沿技术和创新功能而闻名，是 RHEL（Red Hat Enterprise Linux）的上游项目。

## 主要特点

- **前沿技术**：率先采用最新的 Linux 内核和软件包
- **开源纯粹性**：坚持自由开源软件原则
- **创新平台**：许多新技术首先在 Fedora 中测试和部署
- **强大的社区**：活跃的全球开发者和用户社区
- **多种桌面环境**：支持 GNOME、KDE、Xfce 等多种桌面环境

## 安装说明

1. 下载适合 Radxa Dragon Q6A 的 Fedora 镜像
2. 使用镜像写入工具将镜像烧录到 SD 卡或 UFS 存储
3. 将存储介质插入设备并启动
4. 按照屏幕提示完成初始配置
