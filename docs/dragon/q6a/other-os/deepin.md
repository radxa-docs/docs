# deepin

deepin（深度操作系统）是一款基于 Linux 的开源桌面操作系统，由中国武汉深之度科技有限公司（现为统信软件技术有限公司的一部分）主导开发。它以美观的用户界面、良好的开箱即用体验和对中文用户的友好支持而闻名，是中国最具影响力的 Linux 发行版之一。

## 主要特点

- **精美的桌面环境**：采用自主研发的桌面环境（DDE），界面简洁优雅
- **开箱即用**：预装常用软件，降低新手使用门槛
- **中文友好**：完善的中文本地化支持，适合中文用户
- **应用商店**：提供丰富的应用程序，方便软件安装和管理
