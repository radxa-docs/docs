# 蓝牙使用

:::tip 蓝牙功能

瑞莎 NX5 载板板载 M.2 E Key 接口，可用于安装 WiFi/蓝牙模块。

目前测试兼容配件：

- 瑞莎 A8 无线模块
- 英特尔 AX210 无线模块

:::
