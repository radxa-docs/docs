# 核心模组产品线

瑞莎的核心模组（System on Module）产品线覆盖多种形态，方便用户根据应用场景和既有平台灵活选型与扩展。下面的对比表汇总了各系列的封装形态、兼容生态和典型应用，帮助你快速选型：

| 系列 | 形态与封装                                                   | 兼容生态 / 标准                             | 主要特点                                                                           | 典型应用场景                                 |
| ---- | ------------------------------------------------------------ | ------------------------------------------- | ---------------------------------------------------------------------------------- | -------------------------------------------- |
| CM   | 与树莓派 CM4 / CM5 兼容的核心板，采用板对板连接器            | 树莓派 CM4 生态                             | 尺寸与接口兼容树莓派 CM4 / CM5，便于在现有树莓派计算模块方案上直接替换升级计算性能 | 原有 CM4 设备的性能升级、方案快速验证        |
| NX   | Jetson Nano / Orin NX 兼容核心模组，采用 SODIMM 金手指连接器 | NVIDIA Jetson Nano / Orin Nano / Orin NX 等 | 集成大 AI 算力，面向边缘 AI 推理，兼容 Jetson 硬件生态                             | 边缘 AI 网关、机器视觉、智能机器人、工业检测 |

通过这几大系列，瑞莎可在从开发验证到大规模量产的不同阶段，为你提供统一的软件生态与多样化的硬件形态选择。
