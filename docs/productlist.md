# 瑞莎产品列表

欢迎来到瑞莎产品中心！我们专注于迷你主板、计算模块与网络计算设备的研发与制造，产品涵盖 ARM、x86 等多种架构平台，广泛应用于嵌入式开发、多媒体娱乐、人工智能及边缘计算等领域。

- **迷你主板**：信用卡尺寸至掌心大小，涵盖入门级到旗舰级性能，满足从爱好者到工业应用的多样化需求
- **主板**：标准 ATX / ITX 规格，适合桌面级应用与工业扩展场景
- **计算模块**：标准化 SODIMM / B2B 接口，方便快速集成到各类嵌入式产品和工业设备中
- **边缘计算**：专为边缘 AI 和物联网场景设计的迷你计算设备
- **网络计算设备**：多网口设计，专注路由器、防火墙及边缘网关等网络应用场景
