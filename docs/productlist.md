# 瑞莎产品列表

欢迎来到瑞莎产品中心！瑞莎产品覆盖迷你主板、计算模块、网络计算设备、AI 加速模组和边缘计算五大品类，适用于通用计算、嵌入式开发、工业控制、边缘 AI、网络设备和多媒体等多元化场景。

- **迷你主板 / 主板**：适合通用计算、多媒体、物联网和边缘 AI 应用
- **计算模块系列**：适合嵌入式系统、工业控制和定制化产品
- **网络计算设备系列**：适合网络设备、路由器和边缘计算
- **AI 加速模组系列**：面向边缘 AI 推理，支持主流深度学习框架
- **边缘计算设备系列**：开发生态完整，工业级设计，适用于多场景边缘智能部署
