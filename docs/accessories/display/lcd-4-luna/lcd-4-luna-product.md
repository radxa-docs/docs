# 瑞莎 4 Luna 高清触摸屏

## 产品概述

<div style={{textAlign: 'center'}}>
    <img src="/img/accessories/display/display-4-luna-view.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

瑞莎 4 Luna(Display 4 Luna) 是一款带触摸功能的 IPS 显示屏，专为多主板、多场景快速部署而设计。

采用 USB Type-C 接口，单根电缆即可实现视频传输和触控功能，支持扩展或镜像显示模式，可在 Linux、Windows 和 macOS 主机上即插即用，也兼容 Radxa、树莓派等常见单板计算机。该显示屏具有以下特点：

- 4.0 英寸 IPS 液晶显示器
- 720 x 720 分辨率
- 5 点电容式触摸屏
- USB Type-C 连接（视频 + 触控）
- 支持扩展 / 镜像显示模式
- 跨平台支持（Linux / Windows / macOS）

## 产品规格

- 尺寸：4.0 英寸
- 分辨率：720×720
- 显示类型：IPS
- 触控类型：5 点电容触控
- 连接接口：USB Type-C（视频 + 触控）
- 显示模式：扩展 / 镜像
- 操作系统支持：Linux / Windows / macOS
- 供电方式：USB Type-C 供电
- 外壳 (可选)：MagSafe 风格外壳
- 树莓派支持：支持 Raspberry Pi 3 / 4 / 5
- 功耗：典型值 < 1 W
- 亮度：350 nits
- 可视角度：178°（水平 / 垂直）
- 对比度：800:1
