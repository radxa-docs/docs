# 瑞莎 DC12 36W 电源适配器

瑞莎 DC12 36W 电源适配器是专为 Radxa 的 SBC 产品设计的适配器。它可以在 12V 时提供高达 3A 的电流，以满足 Radxa SBC 连接多个外围设备的电源要求。兼容 Radxa ROCK 5T 等 Radxa SBC，以及 Radxa CM3/3S/3I CM4 CM5 底板供电。

![Power DC12 36W](/img/accessories/power/power-dc12-36w.webp)

## 产品规格

- 接口规格：5525，即是外径为 5.5mm，内径为 2.5mm
- 电源输入：100-240V ~ 50/60Hz 0.8A MAX
- 电源输出：12V / 3A 36W
- 具备 UL / CE / FCC / CCC 等认证
- 极性：内正外负

## 配件清单

- 1x 瑞莎 DC12 36W 电源适配器
- 1x 澳规转接头
- 1x 欧规转接头
- 1x 英规转接头
- 1x 美规转接头
