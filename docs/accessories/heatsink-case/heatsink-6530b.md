# 瑞莎 6530B 散热器

## 产品简介

瑞莎 6530B 散热器是一款铝合金材质的主动散热器，适用于为主板提供更稳定的散热能力。

- 高效散热设计
  - 铝合金材质，导热性能优异
  - 立体鳍片结构，有效提升散热面积
- 智能 PWM 调速
  - 实时调节风扇转速
  - 加快空气流通效率，平衡散热性能与运行噪音

- 材质：铝合金

- 适用于：
  - 瑞莎 Cubie A7Z

## 安装指南

以瑞莎 Cubie A7Z 安装 6530B 散热器为例。

### 注意事项

- 安装前请断电，并避免在主板带电时插拔风扇接口。
- 固定螺丝时请对角逐步拧紧，避免散热器受力不均导致与 SoC 接触不良。
- 若导热硅胶片表面带有保护膜，请在贴合前先撕下两侧保护膜。

### 配件清单

- 1x 散热器
- 2x 导热硅胶片
- 4x 螺丝（CM2*4*4）

### 放置导热硅胶

将导热硅胶片对准 SoC 覆盖区域，平整贴合在主板的 SoC 上。

将散热器对准安装孔位放置在主板上，确保散热器底面与导热硅胶片充分贴合。
