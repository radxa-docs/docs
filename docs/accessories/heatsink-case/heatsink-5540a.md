# 瑞莎 5540A 散热器

## 产品简介

瑞莎 5540A 散热器是一款专门用于瑞莎 CM3 和 CM5 的被动散热器。

:::tip 版本说明

瑞莎 5540A 散热器有两个版本，旧版本为银色，新版本为黑色。

:::

## 产品规格

- 材质：铝合金
- 机械尺寸：55 x 40 x 10 毫米
- 适用于：
  - 瑞莎 CM3
  - 瑞莎 CM5

## 安装指南

以瑞莎 CM3 安装 5540A 散热器作为演示。

### 配件清单

- 1x 散热器
- 1x 导热硅胶片
- 4x CM2 x 4 螺丝

### 安装天线（可选）

若自己有外置天线，需要在安装散热器之前安装到 CM3 上。

<div style={{textAlign: 'center'}}>
    <img src="/img/accessories/heatsink-case/heatsink-5540a/install-ante.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### 安装散热硅胶

将导热硅胶片贴在 CM3 的 SoC 上。

<div style={{textAlign: 'center'}}>
    <img src="/img/accessories/heatsink-case/heatsink-5540a/install-silicon.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### 固定散热器

使用 4 颗 CM2 x 4 螺丝从 CM3 的固定孔位底部固定散热器。

<div style={{textAlign: 'center'}}>
    <img src="/img/accessories/heatsink-case/heatsink-5540a/install-heatsink.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### 安装到载板

将散热器和 CM3 整体安装到 CM3 载板上。
