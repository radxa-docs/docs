# 使用散热器

由于树莓派 CM5 性能较强，在长时间高负载运行时会产生较多热量，散热器可以提升散热效果，确保设备稳定运行。

## 安装散热器

参考[安装散热器](./install-heatsink.md)教程完成散热器的安装。

## 温控风扇

### 修改配置文件

编辑 `/boot/firmware/usercfg.txt` 文件，添加以下内容：

```text
dtoverlay=gpio-fan,gpiopin=13,temp=55000
```

参数说明：

- `gpiopin=13`：GPIO引脚号，对应树莓派的 GPIO 13 引脚
- `temp=55000`：温度阈值，单位为毫摄氏度，55000表示55°C

保存文件后，重启设备使配置生效。

### 验证风扇工作

当风扇温度达到配置的阈值时，风扇将自动启动。
