在 Radxa X2L 上集成了 Intel J4125 的 CPU 和一块 [RP2040](https://en.wikipedia.org/wiki/RP2040) MCU，两者通过 USB 和 UART 两种方式进行通信。
RP2040 在系统中以 USB 存储设备的形式存在，程序烧录时只需将文件拖拽至该 USB 设备中即可。
此外，它们还共用 UART 通信接口，使得 Intel J4125 能够通过 UART 控制 RP2040，例如，通过 UART 接口实现对 RP2040 GPIO 的操控。
