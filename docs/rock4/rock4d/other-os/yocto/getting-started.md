# 快速上手

Yocto 项目（YP）是一个开源协作项目，可帮助开发人员创建基于 Linux 的自定义系统。

本节教程目的是引导大家快速上手 Yocto 项目，基于厂商版本的 YP 进行 Linux 系统的构建和开发。

:::tip Yocto 版本

基于瑞芯微 SoC 设计的 Radxa 系列主板 Yocto 项目主要版本为：

- [社区版本](https://github.com/radxa/meta-rockchip/tree/scarthgap)

社区版本的 meta-rockchip 为基于瑞芯微 SoC 的设备提供 OpenEmbedded BSP 支持，并尽可能的使用上游源。

- [厂商版本](https://github.com/radxa/yocto-manifests/tree/scarthgap)

我们基于瑞芯微的厂商版本，增加了 Radxa 部分设备支持，以提供更好的使用体验。

:::
