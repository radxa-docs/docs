# 准备工作

## 开发准备

<Tabs queryString="target">

### 供电

推荐使用 5V/3A Type-C 电源适配器给 ROCK 5C 供电。

:::tip 供电方案

ROCK 5C 主板仅支持 5V 电源输入，兼容 PD 协议。建议电流 3A 以上，确保所有外设稳定运行。

参考电源：

- [瑞莎 PD 30W电源适配器(推荐使用)](https://radxa.com/products/accessories/power-pd-30w)

- 标准 Type-C 电源适配器( 5V 电源输入，支持 PD 协议，建议电流 3A 以上)

:::

### 存储

支持 microSD 卡和 eMMC 模块启动系统。

- microSD 卡：容量建议大于 8GB，用于从 microSD 启动系统系统
- eMMC 模块：容量建议大于 8GB， 用于从 eMMC 模块启动系统

:::tip 推荐配件

- [瑞莎 eMMC 模块](https://radxa.com/products/accessories/emmc-module)

:::

### 读卡器

- microSD 读卡器：用于给 microSD 卡安装系统
- 瑞莎 eMMC 读卡器：用于给 eMMC 模块安装系统

:::tip 推荐配件

eMMC 模块支持线刷安装系统和使用 eMMC 读卡器安装系统，使用 eMMC 读卡器时，安装系统方式与 microSD 卡一致。

- [瑞莎 eMMC/UFS 二合一读卡器](https://radxa.com/products/accessories/emmc-ufs-module-reader)

:::

</TabItem>
