# 第三方 3D 打印外壳

本页收录社区作者为 ROCK 5B+ 设计并开源的第三方 3D 打印外壳，方便用户快速打印并保护主板。

> 说明：以下外壳均为第三方设计，与 Radxa 官方无从属关系，请在打印和使用前自行评估风险。

## 型号一：ROCK 5B+ 外壳

- MakerWorld 模型页面：
  - https://makerworld.com.cn/zh/models/2272559

特点概述：

- 适配 ROCK 5B+ 板卡外形和接口开孔
- 适合作为桌面或实验环境中的日常防护外壳
- 可作为 3D 打印自制外壳的参考方案

---

打印建议：

- 尽量使用耐热、尺寸稳定性较好的材料（如 PETG、ABS 等）
- 建议在打印前阅读模型作者在 MakerWorld 页面中的说明（推荐层高、填充率、支撑设置等）
- 安装时注意避免暴力压迫排线、接口和器件
