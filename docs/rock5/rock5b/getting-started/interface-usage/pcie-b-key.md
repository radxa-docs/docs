# PCIe B Key（仅 ROCK 5B+）

PCIe B Key 可以用来插入 wwan 模块，从而实现蜂窝网络连接功能。

## 支持的配件

[quectel-lte-ec25](../../../../accessories/peripherals/quectel-lte-ec25.md)

## 第三方配件问题记录

1. 使用 EM7430/EM7455 模块时，提示无法识别到 sim 卡，情况可能与 sim 卡识别引脚相关，可按照论坛用户反馈，屏蔽其中一个识别引脚即可正常使用：
[论坛链接](https://forum.radxa.com/t/rock-5b-sim-card-design-flaw/24530)
