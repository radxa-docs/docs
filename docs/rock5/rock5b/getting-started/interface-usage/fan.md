# 风扇接口

## 接口规格

ROCK 5B / ROCK 5B+ 主板的风扇接口适配 **2-Pin、1.25 mm 间距、5V DC** 的风扇连接线。

:::tip 选型参考
如果你使用的是官方散热器，可参考 [瑞莎 6240B 散热器](../../../../accessories/heatsink-case/heatsink-6240b) 的规格说明，其风扇接口为 **2-Pin 1.25 mm 间距连接器**。
:::
