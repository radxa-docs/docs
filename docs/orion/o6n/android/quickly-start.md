# 快速上手

使用安卓系统，需要更新 Android 系统 BIOS 固件，然后将 Android 系统刷入到 NVMe 固态硬盘中。

<BIOS board="星睿 O6N" power="DC 12V 电源适配器" download_page="../../download" tag="o6n" />

## 安装系统

BIOS 固件更新成功后，按照下面教程将最新 Android 系统刷入到 NVMe 固态硬盘中。

- [安装系统到 NVMe 固态硬盘](../getting-started/install-system/nvme-system/nvme-reader)

:::tip 注意事项

将 Android 系统安装到 NVMe 固态硬盘，需要注意以下几点：

- NVMe 固态硬盘容量需要大于等于 128GB
- 安装系统后，将 NVMe 固态硬盘安装到星睿 O6N 正面的 M.2 M Key 2280 插槽中，背面的 M.2 M Key 2280 插槽无法启动 Android 系统

:::
