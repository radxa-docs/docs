# 第三方 3D 打印外壳

本页收录社区作者为 Radxa Orion O6N 设计并开源的第三方 3D 打印外壳，方便用户快速打印并保护主板。

> 说明：以下外壳均为第三方设计，与 Radxa 官方无从属关系，请在打印和使用前自行评估风险。

## 型号一：无螺丝固定外壳

- MakerWorld 模型页面：
  - https://makerworld.com.cn/zh/models/1861722-rui-sha-radxa-orion-o6n-3dda-yin-wai-ke-mian-luo-s

特点概述：

- 适配 Radxa Orion O6N 板卡孔位
- 无螺丝固定设计，通过卡扣设计实现安装和拆卸
- 防尘设计，适合需要长期固定在机柜、工作台等场景

## 型号二：超轻极简保护外壳

- MakerWorld 模型页面：
  - https://makerworld.com.cn/zh/models/1865300-radxa-orion-o6nwai-ke

特点概述：

- 结构相对简洁，便于快速打印
- 适合作为桌面或实验环境中的日常防护外壳

---

打印建议：

- 尽量使用耐热、尺寸稳定性较好的材料（如 PETG、ABS 等）
- 建议在打印前阅读模型作者在 MakerWorld 页面中的说明（推荐层高、填充率、支撑设置等）
- 安装时注意避免暴力压迫排线、接口和器件
