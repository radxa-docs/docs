# 瑞莎 8420B 主动散热器

主要介绍如何安装和拆卸瑞莎 8420B 主动散热器

:::tip

该教程适用于星睿 O6 / O6N，两者安装方式和拆卸方式基本一致，瑞莎 O6 相比 O6N 需要多拆卸一个亚克力外壳。

瑞莎 O6N 可以从亚克力外壳拆卸后，再按照教程操作进行安装和拆卸。

:::

## 安装 8420B 主动散热器

### 安装 8420B 主动散热器

#### 涂抹导热硅脂

挤出适量导热硅脂涂抹至主板 SOC 芯片上。

<div style={{textAlign: 'center'}}>
  <img src="/img/orion/o6/heatsink-8420b/8240b-assemble-01.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

#### 固定散热器

使用螺丝刀以对角交替松动的方式拧紧 4 颗散热器的固定螺丝，确保均匀受力。

<div style={{textAlign: 'center'}}>
  <img src="/img/orion/o6/heatsink-8420b/8240b-assemble-02.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### 连接散热器排线

将散热器排线连接到主板对应的接口上。

<div style={{textAlign: 'center'}}>
  <img src="/img/orion/o6/heatsink-8420b/8240b-assemble-03.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### 安装亚克力外壳

## 拆卸 8420B 主动散热器

### 拆卸亚克力外壳

:::tip
适用于购买了亚克力外壳的用户，若没有购买亚克力外壳，可以跳过该步骤！
:::

用螺丝刀将固定上下亚克力盖板的螺丝拧下，取下固定亚克力板的铜柱和亚克力板。

<div style={{textAlign: 'center'}}>
  <img src="/img/orion/o6/heatsink-8420b/8240b-disassemble-01.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

### 拆卸 8420B 主动散热器

#### 断开散热器排线

断开散热器和主板的排线连接。

<div style={{textAlign: 'center'}}>
  <img src="/img/orion/o6/heatsink-8420b/8240b-disassemble-02.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>

#### 拆卸散热器

使用螺丝刀以对角交替松动的方式拧开 4 颗散热器的固定螺丝，确保均匀受力。

<div style={{textAlign: 'center'}}>
  <img src="/img/orion/o6/heatsink-8420b/8240b-disassemble-03.webp" style={{width: '100%', maxWidth: '1200px'}} />
</div>
