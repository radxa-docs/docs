# 性能基准测试

Benchmark 是了解硬件平台网络模型运行速度的最佳途径。

:::info
基准测试结果可能因具体应用场景和模型优化程度而有所不同，仅供参考
:::

## 测试说明

- 测试工具：axcl_run_model
- Batch Size：1 or 8
- 单位：FPS（Frame/Second）

:::info
由于不同 Host 其 memcopy、pcie 性能差异，因此 `axcl_run_model` 只统计网络模型在 Device 上的推理耗时
:::
